# YouTube评论抓取器项目

## 项目概述

这是一个功能完整的YouTube视频评论抓取工具，能够自动抓取YouTube视频下的所有评论，包括回复。项目使用Python开发，基于Selenium WebDriver技术，支持自动滚动、展开回复，并将结果保存为多种格式。

## 核心功能

### ✅ 已实现的功能

1. **完整评论抓取**
   - 抓取视频下的所有主评论
   - 自动展开并抓取所有回复评论
   - 支持深层回复的递归抓取

2. **智能页面操作**
   - 自动滚动页面加载更多评论
   - 智能点击"显示回复"按钮
   - 动态检测页面加载状态

3. **多格式输出**
   - JSON格式：结构化数据，便于程序处理
   - CSV格式：适合Excel等工具分析
   - TXT格式：人类可读的文本格式

4. **数据完整性**
   - 评论者用户名/ID
   - 评论发布时间
   - 评论完整内容
   - 是否为回复的标识
   - 数据抓取时间戳

5. **用户体验优化**
   - 无头模式支持（后台运行）
   - 可配置的滚动间隔
   - 详细的进度提示
   - 错误处理和恢复

6. **自动化部署**
   - 自动ChromeDriver管理
   - 依赖包自动安装
   - 跨平台兼容性

## 安装和使用

### 快速开始
```bash
# 1. 克隆或下载项目文件
# 2. 安装依赖
python3 quick_setup.py

# 3. 运行程序
python3 youtube_comment_scraper_improved.py "YouTube视频URL"
```

### 高级用法
```bash
# 无头模式，JSON格式输出
python3 youtube_comment_scraper_improved.py "URL" --headless --format json

# 自定义滚动间隔和输出文件名
python3 youtube_comment_scraper_improved.py "URL" --scroll-pause 3 --output "my_data"
```

## 故障排除

### 常见问题
1. **Chrome驱动问题**: 使用webdriver-manager自动解决
2. **网络超时**: 增加滚动间隔时间
3. **内存不足**: 使用无头模式或增加系统内存
4. **权限问题**: 使用--break-system-packages安装依赖
