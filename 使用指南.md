## 详细使用方法

### 命令行参数说明

```bash
python3 youtube_comment_scraper_improved.py [YouTube视频URL] [选项]
```

**必需参数：**
- `url`: YouTube视频的完整URL

**可选参数：**
- `--format`: 输出格式，可选值：`json`、`csv`、`txt`、`all`（默认：`all`）
- `--headless`: 使用无头模式运行（不显示浏览器窗口）
- `--scroll-pause`: 滚动间隔时间，单位秒（默认：2秒）
- `--output`: 自定义输出文件名（不含扩展名）

### 使用示例

#### 1. 基础抓取
```bash
python3 youtube_comment_scraper_improved.py "https://www.youtube.com/watch?v=dQw4w9WgXcQ"
```
这将抓取视频的所有评论，并保存为JSON、CSV、TXT三种格式。

#### 2. 指定输出格式
```bash
# 只保存为JSON格式
python3 youtube_comment_scraper_improved.py "VIDEO_URL" --format json

# 只保存为CSV格式
python3 youtube_comment_scraper_improved.py "VIDEO_URL" --format csv
```

#### 3. 使用无头模式（推荐）
```bash
python3 youtube_comment_scraper_improved.py "VIDEO_URL" --headless
```
无头模式运行更快，占用资源更少，适合批量处理。

#### 4. 自定义文件名
```bash
python3 youtube_comment_scraper_improved.py "VIDEO_URL" --output "我的评论数据"
```
这将生成：`我的评论数据.json`、`我的评论数据.csv`、`我的评论数据.txt`

#### 5. 调整滚动速度
```bash
# 对于网络较慢的情况，增加滚动间隔
python3 youtube_comment_scraper_improved.py "VIDEO_URL" --scroll-pause 5
```

### 编程接口使用

```python
from youtube_comment_scraper_improved import YouTubeCommentScraperImproved

# 创建抓取器实例
scraper = YouTubeCommentScraperImproved(
    headless=True,          # 使用无头模式
    scroll_pause_time=2     # 滚动间隔2秒
)

try:
    # 抓取评论
    video_url = "https://www.youtube.com/watch?v=VIDEO_ID"
    comments = scraper.scrape_comments(video_url)
    
    print(f"抓取到 {len(comments)} 条评论")
    
    # 保存结果
    scraper.save_to_json("comments.json")
    scraper.save_to_csv("comments.csv")
    scraper.save_to_txt("comments.txt")
    
    # 处理评论数据
    for comment in comments:
        print(f"作者: {comment['author']}")
        print(f"时间: {comment['time']}")
        print(f"内容: {comment['content']}")
        print(f"是否为回复: {comment['is_reply']}")
        print("-" * 30)
        
finally:
    # 关闭浏览器
    scraper.close()
```

## 输出文件格式

### JSON格式
```json
{
  "total_comments": 150,
  "extracted_at": "2024-01-01T12:00:00",
  "comments": [
    {
      "author": "用户名",
      "time": "2天前",
      "content": "这是评论内容",
      "is_reply": false,
      "extracted_at": "2024-01-01T12:00:00"
    }
  ]
}
```

### CSV格式
包含以下列：
- `author`: 评论者用户名
- `time`: 评论时间
- `content`: 评论内容
- `is_reply`: 是否为回复（True/False）
- `extracted_at`: 抓取时间

### TXT格式
人类可读的文本格式，回复评论会有缩进显示。

## 常见问题解决

### Q1: 提示"Chrome驱动初始化失败"
**解决方案：**
1. 确保已安装Chrome浏览器
2. 程序会自动下载ChromeDriver，请确保网络连接正常
3. 如果仍有问题，尝试手动安装Chrome：
```bash
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb
```

### Q2: 抓取的评论数量很少
**可能原因及解决方案：**
1. **网络慢**：增加滚动间隔时间
```bash
python3 youtube_comment_scraper_improved.py "URL" --scroll-pause 5
```

2. **视频评论较少**：正常现象

3. **页面加载问题**：尝试多次运行

### Q3: 程序运行很慢
**优化建议：**
1. 使用无头模式：`--headless`
2. 减少滚动间隔：`--scroll-pause 1`
3. 关闭其他占用内存的程序

### Q4: 无法抓取某些评论
**可能原因：**
1. YouTube页面结构变化（程序需要更新）
2. 某些评论被隐藏或限制
3. 网络连接不稳定

### Q5: 出现"externally-managed-environment"错误
**解决方案：**
使用 `--break-system-packages` 选项：
```bash
pip3 install --break-system-packages selenium webdriver-manager requests
```

```python
scraper = YouTubeCommentScraperImproved(headless=True)

try:
    for i, url in enumerate(urls):
        print(f"处理第 {i+1}/{len(urls)} 个视频")
        comments = scraper.scrape_comments(url)
        scraper.save_to_json(f"video_{i+1}_comments.json")
        
        # 添加延迟避免请求过于频繁
        time.sleep(10)
        
finally:
    scraper.close()
```
